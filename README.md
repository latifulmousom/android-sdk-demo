#### Sawo Passless Demo For Android Devices

##### Testing the demo

Clone the project, import the project in Android Studio and in MainActivity add an API Key and API key secret

Or, download the release and install it in your device


##### Testing trusted device functionality

1. Download the release apk, login with normal OTP flow
2. Now go to [http://sawo-custompilot.herokuapp.com/](http://sawo-custompilot.herokuapp.com/) and
 try to login with same identifier you used to login in apk, you should get a notification in your device
3. Open the notification and allow it, you will be logged in the web app
